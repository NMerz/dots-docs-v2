Get a `transaction` object by its `id`.

## Retrieve a Transaction

### Method

Make a request to [/v2/transactions/get/transaction/{TRANSACTION_ID}](/api#tag/transactions/paths/~1api~1transactions~1get~1transaction~1%7Btransaction%5Fid%7D/get)
